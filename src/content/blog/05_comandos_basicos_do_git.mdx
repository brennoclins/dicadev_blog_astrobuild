### Principais comandos _git_

```bash
git init
git add .
git status
git commit -m “chore: Commit Inicial”
git push
```

### Baixando um projeto do github na minha maquina

```bash
git clone <endereço do repositório>
cd <nome do projeto>
```

### Criando uma branch nova

```bash
git checkout -b feature/test-novo-driver
```

### Mudar de branch

```bash
git checkout main
```

### Visualizar os commits realizados

```bash
git log
```

### Para saber qual o endereço do servidor remoto do projeto

```bash
git remote -v
```

### Para sincronizar os repositórios do GitHub com o repositório em sua maquina local

```bash
git pull
```
